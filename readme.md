# Modeling of dependency of equities and foreign exchange using copula model

## Running the code

### Prerequisites

- `python 3`
- `CLI environment`
  - widows (Powershell/Command prompt)
  - Linux & Unix (terminal)
- `python PIP`

#### Required modules

- pandas
- scipy
- copulas
- matplotlib
- numpy

Install using command

```sh
pip install pandas scipy copulas matplotlib numpy
```

### Command to run the code

```sh
python app.py
```
